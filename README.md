# **Shubhankar Gupta — Portfolio**

## **Tech Stack**

* React
* Vite
* React Router
* Tailwind CSS
* Framer Motion
* Vercel (Deployment)


## **Directory Structure**

```
src/
  components/
    ui/
    layout/
    sections/
  pages/
  hooks/
  data/
  assets/
  styles/
  lib/
public/
```


## **Getting Started**

Clone the repository:

```
git clone <repo-url>
cd portfolio
```

Install dependencies:

```
npm install
```

Run the dev server:

```
npm run dev
```

Create a production build:

```
npm run build
```

Preview the production build:

```
npm run preview
```

## **Deploying to Vercel**

* Push the repository to GitHub
* Import the project into Vercel
* Select the Vite framework preset
* Build command: `npm run build`
* Output directory: `dist`

Vercel handles the rest.
